نسخة معدلة حسب الطلب:
- مزامنة Firebase ثابتة على مفتاح الشركة وتعمل تلقائياً بعد الحفظ.
- تغيير كلمة مرور المدير يحفظ محلياً ويرفع إلى Firebase.
- دخول الموظف يعمل باسم المستخدم أو البريد أو الاسم.
- إضافة الصنف: الكرتونة ثابتة، سعر شراء/بيع الكرتونة، وحساب سعر الوحدة تلقائياً.
- الكاشير يدعم وحدة/كرتونة/كسور وتعديل السعر عند البيع.
- الفواتير: عرض/تعديل/طباعة/PDF/تحميل PNG وحذف مع عكس الحسابات والمخزون.
- الديون: سجل دين، دين يدوي، تطبيق لاحق، دفعات سداد بحساب مختار، وإجمالي لكل عميل.
- تحسين التصميم والأيقونات والاستجابة للجوال.
