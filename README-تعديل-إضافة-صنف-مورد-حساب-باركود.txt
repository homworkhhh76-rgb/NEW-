تعديل صفحة إضافة صنف - 2026-05-09

تم تنفيذ المطلوب في صفحة: إضافة-صنف.html

التغييرات:
1) إضافة قائمة الموردين في نموذج إضافة الصنف، وهي اختيارية.
2) إضافة قائمة الحسابات في نموذج إضافة الصنف، وهي اختيارية.
3) عند اختيار حساب يتم خصم إجمالي تكلفة الإدخال من الحساب.
4) إجمالي التكلفة يتم احتسابه حسب وحدة الإدخال:
   - إذا كانت الوحدة كرتونة: الكمية × سعر شراء الكرتونة.
   - إذا كانت الوحدة وحدة/كيلو/لتر: الكمية × سعر شراء الوحدة.
5) المخزون يتم حفظه واحتسابه بالوحدات دائمًا:
   - كرتونة: كمية الكراتين × عدد الوحدات في الكرتونة.
   - وحدة/كيلو/لتر: الكمية كما هي.
6) عند إدخال باركود موجود سابقًا:
   - لا يتم إنشاء صنف جديد.
   - يتم زيادة مخزون الصنف السابق بالكمية الجديدة.
   - يتم تحديث اسم الصنف إلى الاسم الجديد إذا تغيّر.
   - يتم تحديث سعر الشراء والبيع إلى السعر الجديد إذا تغيّر.
   - لا يتم تعديل أو العبث بسجلات الحسابات أو الفواتير القديمة.
7) يتم تسجيل حركة مخزون توضح الزيادة عند تحديث صنف موجود.

ملاحظات:
- السجلات القديمة تبقى كما هي لأنها محفوظة كسجلات منفصلة وقت العملية.
- حركة الحساب الجديدة يتم إنشاؤها فقط عند اختيار حساب ووجود تكلفة أكبر من صفر.
