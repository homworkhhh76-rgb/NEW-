نسخة نهائية معدلة حسب آخر طلب:
- تصميم أخضر حديث شبيه بتطبيق جوال مع خط Tajawal.
- زر تسجيل خروج أعلى الصفحة.
- العملاء يعرضون الديون، والضغط على العميل يفتح سجل الديون والفواتير والحركات.
- صفحة الديون مرتبطة بالعملاء وتدعم دين يدوي وسداد.
- الكاشير افتراضيًا يضيف الصنف بوحدة، والكرتونة اختيار ثانٍ.
- إضافة صنف: عند اختيار كرتونة يتم إدخال سعر الشراء للكرتونة وسعر البيع للكرتونة ويحسب سعر الوحدة تلقائيًا.
- الفواتير قابلة للطباعة/PDF وصورة PNG والتعديل والحذف مع إعادة الحسابات.
- تقرير الأرباح افتراضيًا يحسب ربح فرق السعر مع خصم المصاريف والأجور، ويوجد خيار آخر للمبيعات-المشتريات.
- مزامنة Firebase تلقائية بعد الحفظ ومع زر مزامنة دوّار، وسحب دوري للبيانات.
- دخول الموظف صار يقبل اسم المستخدم أو البريد أو الاسم مع كلمة المرور، ويسحب بيانات الشركة من Firebase قبل التحقق.
