# مشروع سوبرماركت - محاسبة ومخزون

## التشغيل
افتح `index.html` أو ارفع المجلد كاملًا على الاستضافة.

- كلمة مرور المدير الافتراضية: `1234`
- كل صفحة HTML مستقلة وبداخلها CSS و JavaScript.
- ملف Firebase الوحيد هو: `firebase.js`.
- يعمل أوفلاين بعد أول تحميل بفضل Service Worker والتخزين المحلي.
- عند عودة الإنترنت يمكن الضغط على زر المزامنة أعلى الصفحة.

## مهم
إذا كان رابط Realtime Database مختلفًا، عدّل `databaseURL` داخل `firebase.js`.

## ملاحظات
هذه نسخة تأسيسية كبيرة قابلة للتطوير: الصفحات الأساسية موجودة ومربوطة محليًا، والحسابات تسجل وارد/صادر ورصيد قبل/بعد.
